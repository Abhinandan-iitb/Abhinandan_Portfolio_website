# Abhinandan's Portfolio Website

Welcome to my portfolio website! This site showcases my personal and professional details, educational qualifications, projects, and blogs. It also provides contact links to my LinkedIn and Gmail.

# Features
- Personal and Professional Details: An overview of my background.
- Educational Qualifications: Details about my academic journey.
- Projects: A showcase of my work and contributions.
- Blogs: A section where I share my thoughts and experiences.
- Contact Links: Easily connect with me on LinkedIn and via Gmai.

# GitHub Repository

You can access the project repository here: (https://github.com/Abhinandan-iitb/Abhinandan_Portfolio_website)

# Clone the Repository

You can clone the repository using the following command:
git clone https://github.com/Abhinandan-iitb/Abhinandan_Portfolio_website.git

Navigate to the Project Directory
cd flask_hw_10

Simply run the Python file to start the application:
python3 flask_h_10.py

After running the Flask app, open your browser and click on the link:
http://localhost:5000/
